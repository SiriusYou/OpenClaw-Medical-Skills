一、项目概述

什么是 OpenClaw Medical Skills？

OpenClaw Medical Skills 是一个为 OpenClaw / NanoClaw 框架设计的、经过精心策划的开源医学 AI 技能集合。 它汇集了来自 12+ 个开源技能仓库 的 869 个技能，覆盖生物医学与临床研究的完整领域。 每个技能都是一个自包含的模块（核心为 SKILL.md 文件），赋予通用 AI Agent 专业的医学与科学研究能力。


核心价值：

  • 真实的 PubMed、ClinicalTrials.gov、FDA 数据库查询（非泛化回答）
  • 完整的 RNA-seq、scRNA-seq、GWAS、变异分析生物信息学流水线
  • 药物发现智能（ChEMBL、DrugBank、DDI 预测、药物警戒）
  • 临床文档撰写（SOAP 病历、出院小结、先验授权审核）
  • 基因组学支持（VCF 注释、ACMG 分类、多基因风险评分）
  • 法规合规指导（FDA、CE 标志、IEC 62304、ISO 14971）

五、核心技能展示

临床与医学

pubmed-search

搜索 PubMed 科学文献。当用户需要查找论文、搜索文献、查找研究成果时自动触发。

无需认证公共 API

medical-research-toolkit

查询 14+ 个生物医学数据库（ChEMBL、PubMed、ClinicalTrials.gov、OpenTargets、OpenFDA、OMIM、Reactome、KEGG、UniProt 等），通过统一 MCP 端点实现。

MCP 端点14+ 数据库

clinical-reports

撰写全面的临床报告 — 病例报告 (CARE)、诊断报告、临床试验报告 (ICH-E3, CSR)、患者文档 (SOAP, H&P)。符合 HIPAA/FDA/ICH-GCP 规范。

HIPAA 合规FDA

tooluniverse-precision-oncology

基于分子谱为癌症患者提供可操作的治疗建议。解读肿瘤突变，识别 FDA 批准的疗法、临床试验和耐药机制。

精准医学

生物信息学与基因组学

bio-variant-calling

基于 GATK 的胚系变异检测流水线，从比对后的 BAM/CRAM 文件开始处理。

GATK流水线

tooluniverse-single-cell

使用 scanpy 进行单细胞 RNA-seq 分析。包含 QC、标准化、PCA、UMAP、Leiden 聚类、轨迹分析和细胞类型注释。

scanpy

alphafold

使用 AlphaFold2 结构预测验证蛋白质设计。验证设计序列是否正确折叠。

AI 预测蛋白质结构

antibody-design-agent

抗体工程设计 Agent，集成 MAGE 和 RFdiffusion 方法。用于治疗性抗体的从头设计与优化。

蛋白质设计

药物发现与安全

tooluniverse-drug-repurposing

利用靶点驱动、化合物驱动和疾病驱动策略识别药物再利用候选物。通过分析靶点、生物活性和安全性概况寻找新适应症。

药物再利用

tooluniverse-drug-drug-interaction

药物-药物相互作用预测与风险评估。分析 CYP450/转运体机制，支持多药联合分析 (3+ 药物)。

DDICYP450

tooluniverse-adverse-event-detection

使用 FDA FAERS 数据、药物标签和生物医学证据检测和分析药物不良事件信号。生成定量安全信号评分 (0-100)。

药物警戒

molecular-dynamics

使用 OpenMM 和 MDAnalysis 运行和分析分子动力学模拟。蛋白质/小分子系统、力场、RMSD/RMSF 分析。

模拟

十一、总结与评价

优势

  • 规模最大 — 869 个医学 AI 技能，覆盖完整生物医学领域
  • 免认证 — 绝大多数技能使用公共 API，零配置即用
  • 模块化 — 每个技能自包含，可按需安装
  • 规范化 — 统一的 SKILL.md 格式，CI 自动验证
  • 多来源 — 聚合 12+ 开源仓库，社区驱动
  • 文档齐全 — 双语文档，48% 技能含使用指南

注意事项

  • 体积大 — 全量安装含 869 个目录，适合选择性安装
  • 依赖外部 API — 需要网络访问才能发挥核心功能
  • Python 为主 — 所有代码示例均为 Python
  • 需配合框架 — 设计用于 OpenClaw/NanoClaw，非独立运行
  • 部分技能需认证 — COSMIC 等少数数据库需注册
